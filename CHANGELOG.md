## [1.0.0] - 2026-04-04

### Added

- Initial release of Heildamm
- Four architectural patterns: feature-based, layer-based, domain-driven, and monorepo
- Four technology stack variants: bare, tRPC, Prisma, and full
- Interactive CLI with colored output and progress indication
- TypeScript support with strict configuration
- Tailwind CSS integration in all templates
- Example implementations in all templates
- Comprehensive project documentation
- Code of Conduct and Contributing guidelines
- Support and security policy documentation
- GitHub issue and pull request templates

### Features

#### CLI

- Interactive project configuration
- Support for current directory or new subfolder creation
- Clear user prompts and visual feedback
- Progress indication during project creation
- Helpful success messages with next steps

#### Architectures

- Feature-based structure for component-driven development
- Layer-based structure for traditional layered architecture
- Domain-driven design structure for complex business domains
- Monorepo structure with pnpm workspaces

#### Technology Stacks

- Bare: Next.js 14, TypeScript, Tailwind CSS
- tRPC: Bare + tRPC v11, Zod validation
- Prisma: Bare + Prisma ORM, PostgreSQL setup
- Full: Complete stack with all technologies integrated

#### Development

- Development server with hot reload
- Production build optimization
- ESLint configuration for code quality
- TypeScript strict mode
- Tailwind CSS configuration with utility-first approach

### Documentation

- Comprehensive README with architecture guides
- Contributing guidelines
- Code of Conduct
- Security policy
- Support documentation
